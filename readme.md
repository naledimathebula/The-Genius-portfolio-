# Personal Developer Portfolio

## 📌 Project Overview

This project is a fully responsive personal developer portfolio website created using **HTML5**, **CSS3**, and **Bootstrap 5**. The portfolio showcases my professional profile, technical skills, completed projects, and contact information while following accessibility and responsive design best practices.

The website was built according to the assignment requirements and includes a Hero Section, About Me Card, Projects Grid, Contact Section, Footer, and a Bonus Feature for automatic dark mode support.

# 🌙 Bonus Feature

## Automatic Dark Mode

Implemented using:

```css
@media (prefers-color-scheme: dark)
```

The portfolio automatically adjusts based on the user's system preference.
